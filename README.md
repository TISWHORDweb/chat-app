# chat-app
CHAT APP with MERN stack (nodejs, mongodb, expressjs, reactjs)

### setup : 
```
  git clone URL
  cd project
  cd server && yarn
  cd client && yarn
  
  // in server 
  yarn start
  
  // in client 
  yarn start
  
 // http://localhost:3000 :)
```

## ----------------------------------
- auth with jsonwebtoken (jwt)
- database with mongodb
- live status
- live messages
- notif if u have new msg
- block user
- more ..
